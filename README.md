# 🏦 Sistema Bancário em Java

## 📌 Descrição
Sistema bancário simples desenvolvido em Java, permitindo operações básicas como consulta de saldo, depósito e saque.

## 🚀 Funcionalidades
- Ver saldo
- Depositar dinheiro
- Sacar com validação de saldo

## 🛠️ Tecnologias
- Java

## 📚 Aprendizados
- Estruturas condicionais
- Controle de fluxo
- Interação com usuário
- Lógica de programação

## ▶️ Como executar
Compile: javac SistemaBanco.java

Execute: java SistemaBanco.java
